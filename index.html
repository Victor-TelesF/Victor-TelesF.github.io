02. Portfólio

Projeto em Destaque

FinTrack API

Python 3.13 FastAPI PostgreSQL Docker
110+ Testes Unitários

Uma API robusta para gestão de carteiras de investimento diversificadas. O sistema lida com o cálculo complexo de rentabilidade para diferentes classes de ativos (Renda Fixa, Variável, Cripto) usando indexadores dinâmicos (CDI, IPCA, Selic).

Arquitetura & Design

  • Domain-Driven: Lógica de negócio 100% isolada de frameworks.
  • Design Patterns: Uso de Strategy e Factory para cálculos polimórficos.
  • Imutabilidade: Estado da carteira derivado puramente de transações.

Engenharia

  • Tipagem Forte: Uso extensivo de Pydantic v2 e Python Typing.
  • Precisão: Cálculos financeiros utilizando o tipo Decimal.
  • Extensibilidade: Contratos via Protocol para inversão de dependência.
