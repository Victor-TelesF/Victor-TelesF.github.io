Projeto em destaque

FinTrack API em desenvolvimento — Etapa 1 concluída

API REST para gestão de carteiras de investimento, com suporte a múltiplas classes de ativos — renda fixa, renda variável e criptomoedas — cada uma com regras de rentabilidade e indexadores próprios (CDI, IPCA, Selic, prefixado). Investidores com carteiras diversificadas precisam consolidar rentabilidades calculadas de formas distintas; o FinTrack resolve isso com uma camada de domínio que trata cada ativo de forma polimórfica, através de um contrato único de cálculo de retorno.

Arquitetura

  • Separação estrita entre domínio (Python puro) e infraestrutura — domain/ nunca importa nada de framework ou banco
  • Hierarquia polimórfica de ativos: Asset (ABC) → FixedIncome / VariableIncome → CDB, GovernmentBond, Stock, RealEstateFund, Cryptocurrency
  • Contrato unificado ReturnContext / MarketRates, garantindo LSP em toda a hierarquia

Decisões de engenharia

  • Strategy Pattern para desacoplar o cálculo de rentabilidade do indexador (cada um com fórmula própria)
  • Factory/Registry resolve automaticamente a estratégia correta a partir do indexador, evitando combinações inválidas
  • Template Method centraliza validação comum entre ativos de renda variável
  • Protocol para contratos tipados sem acoplamento por herança
  • Decimal em todo o domínio — float não é aceitável em cálculo financeiro
